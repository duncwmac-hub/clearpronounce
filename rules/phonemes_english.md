# English phoneme inventory (ClearPronounce E)

All symbols are standard IPA. Voiceless "sh" (as in *she*) is **U+0283 ʃ** (LATIN SMALL LETTER ESH). Use an IPA-friendly font so it displays as the long-s shape, not a jagged form.

Target: rhotic General American-ish, ELF-oriented. R-colored vowels are treated as vowel + /r/ sequences.

## Consonants

/p, b, t, d, k, g, tʃ, dʒ, f, v, θ, ð, s, z, ʃ, ʒ, h, m, n, ŋ, l, r, j, w/

## Vowels (monophthongs)

/i, ɪ, e, ɛ, æ, ɑ, ɔ, ʊ, u, ʌ, ə, ɝ/

- /ɝ/ is the stressed "er" vowel (as in *bird*).

## Diphthongs

/eɪ, aɪ, ɔɪ, aʊ, oʊ/

- R-colored vowels are represented as vowel + /r/ (e.g. /ɪr, ɛr, ɑr, ɔr, ʊr/).
