# ClearPronounce

**Live app:** [clearpronounce.com](https://clearpronounce.com)

This repository is for **ClearPronounce** product and business assets only. It is separate from your personal operating system (`~/DuncanOS`) so that everything here is safe to share with collaborators, schools, or investors.

## What lives here

- Product and curriculum content (e.g. Unit Zero materials, pronunciation guides, worksheets).
- Client-facing documents (school explainers, pilot proposals, pitch decks, brochures).
- Code and automation that directly support ClearPronounce as a product (e.g. generation scripts, website code).
- Documentation in `docs/`, including:
  - `docs/schools_explainer.md` – shareable explainer for language schools.

## What stays in DuncanOS

Keep the following **out of this repo** and in `~/DuncanOS`:

- Personal GTD lists, capture/inbox items, and session logs.
- Personal finances, health notes, or life admin.
- Raw thinking and emotional reactions about ClearPronounce (project files and notes).

For more detail on the boundary between `~/DuncanOS` and this repo, see `CONTRIBUTING.md` and `~/DuncanOS/protocols/clearpronounce_repo_and_backup.md`.
